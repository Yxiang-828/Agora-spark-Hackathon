# Agora — brand

**Product:** Agora · **Studio:** Aegis · **Repo codename:** growth-agent

## Logo
A gateway **arch** (the agora) over an **assembly of three nodes** — the flanking
nodes are people, the **accented center node is the AI**. Humans and their AIs,
gathered in one place.

| File | Use |
|---|---|
| `agora-mark.svg` | the symbol alone, light marks, transparent (on dark surfaces) |
| `agora-icon.svg` | mark on an ink tile — favicon / app icon / plugin icon |
| `agora-logo.svg` | horizontal lockup (mark + wordmark), light, transparent (dark UI) |
| `agora-logo-card.svg` | lockup on an ink panel — login brand image (reads on any bg) |

## Palette
| Token | Hex | Use |
|---|---|---|
| Ink | `#0E1016` | backgrounds, tiles |
| Marble | `#ECE7DD` | primary marks/text on ink |
| Amber | `#E0A15E` | accent (the AI node, highlights) |

## Type
- **Wordmark:** high-contrast serif — currently `Georgia` (system, for reliable
  rasterization). Upgrade to **Fraunces** converted-to-paths for the final lockup
  (matches the workspace portfolio aesthetic).
- **UI:** inherit the app/system stack.

## Rules
- Light marks on dark; on light backgrounds use the `-card` / `-icon` (self-contained ink).
- Keep clear space ≈ the node diameter around the lockup. Don't recolor the arch.
- The amber node is the only accent — use sparingly.
